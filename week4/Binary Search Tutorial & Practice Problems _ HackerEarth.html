Tutorial on Binary Search

by Anand Jaisingh

Binary Search is the most popular Search Algorithm of all times and one of the most commonly used techniques used to solve problems.

It is the most efficient searching algorithm that exists till date. If all the names in the world are written down together in order and you want to search for the position of a particular name, binary search would do it in at max 3535 iterations.

The first thing to know is that Binary Search works only on a sorted set of elements. In order to use binary search on a collection, it first needs to be sorted.

Now if it is required to perform operations on a sorted set, The number of iterations can always be reduced on the basis of the value being searched, isn't it?

Let's consider the following array:

Find the position of element 88 in this array. Linear Search will find the element in 9th step.

Let's see how the number of iterations can be reduced using Binary Search . We need to know the start and end of the range to start the search. Lets call them Low and High.

Low = 0
High = n-1

Now, we need to compare the search value KK to the element located at the median of the lower and upper bound. If our value KK is greater, we can always increase the lower bound, else we need to decrease the upper bound.

Referring to the image above, our lower bound is 00 and the upper bound is 99. Thus, the median of the lower and upper bound is (lower_bound + upper_bound) / 2 = 5. Here a[5] = 4. As we can see the value 4 is greater than the value 2 we are searching for. Thus, we do not need to conduct a search on any element beyond 10 as the elements beyond it will obviously be greater than it.

Thus, we can always drop the upper bound of the array to the position of element 4. Now the same procedure is carried out on this array with :

Low:   0
High:  5

This procedure is repeated recursively until we reach a point when Low = High. This position when Low = High is the position of the element we are searching for in the array.

Implementation:

int binarysearch(int value,int low,int high)
{
    if(low==high)

        retutrn low;
    }
    else
    {
        int mid=(low+high)>>1;
        if(value<a[mid])    
        {
            high=mid;
            return binarysearch(value,low,high);
        }
        else
        {
            low=mid;
            return binarysearch(value,low,high);   
        }
    }
}

You have been given an array AA consisting of NN integers. All the elements in this array AA are unique. You have to answer some queries based on the elements of this array. Each query will consist of a single integer xx. You need to print the rank based position of this element in this array considering that the array is 11 indexed. The rank based position of an element in an array is its position when the array has been sorted.

Note: It is guaranteed that all the elements in this array are unique and for each xx belonging to a query, value ′x′′x′ shall exist in the array

Input Format

The first line consists of a single integer NN denoting the size of array AA. The next line contains NN unique integers, denoting the content of array AA. The next line contains a single integer qq denoting the number of queries. Each of the next qq lines contains a single integer xx denoting the element whose rank based position needs to be printed.

Output Format

You need to print qq integers denoting the answer to each query.
